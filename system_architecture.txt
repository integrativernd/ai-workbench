# AI Workbench

AI Workbench is framework for building semi-autonomous AI agents.

### Tech Stack

- Python 3
- Django
- Postgresql
- Redis
- Selenium
- Heroku
- Anthropic

### Features

- Discord Bot
- Django Admin
- Redis Queue for background tasks
- Google Search
- Web Browsing

### Core Innovations

AI workbench relies heavily on Anthropic's tool API and the instructor library to help
AI agents make decisions.
